# HTML Tables Project

This project demonstrates the use of **HTML Tables** with proper structure, formatting tags, and basic styling. It is useful for beginners who want to understand how to create and manage tables in HTML.

## 📌 Features

- Create simple and structured HTML tables
- Use of rows and columns
- Table headings with `<th>`
- Table data using `<td>`
- Table rows using `<tr>`
- Adding borders to tables
- Merging cells using `colspan` and `rowspan`
- Formatting table content with HTML tags
- Clean and beginner-friendly code

## 🛠 Technologies Used

- HTML5
- CSS3 (optional for styling)
